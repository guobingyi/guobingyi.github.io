iTerm2分屏切换

iTerm2分屏切换

积极的蝙蝠心态，活到老，学到老。

1. 新建窗口

command + t

2. 垂直分屏

command + d

3. 水平分屏

command + shift + d

4. 在最近使用的分屏直接切换

command + ]  OR command + [

5. 切换到指定位置的分屏

command + alt + 方向键

6. 切换标签页

command + 数字

7. 按方向切换标签页

command + 方向键

8. 保存当前窗口快照

shift + command + s

9. 快照回放

command + alt + b

如果不能正常使用，请自行查阅文档。
