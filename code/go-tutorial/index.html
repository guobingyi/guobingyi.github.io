Go Tutorial

学习一门语言最好的方式，就是熟读其文档。

0x00 安装

安装go语言是件轻松的事情，如果安装都非常困难，那么还有什么好说的。

Mac下安装

brew install go

Windows下安装

推荐使用MSI installer方式安装，一直Next下去就可以了。
