Android

记一次Android开发

0x00 记录遇到的问题

  1. 提示:opendir failed, Permission denied

使用run-as 你的包名称进行处理，例如：

run-as com.gby.cleanapp
  1. 关于/system/bin/sh: su: not found的解决办法

Android Studio带(Google Play)的模拟器无法获得root权限安装,换成为带(Google APIs)的模拟器即可。

  1. 想使用Android Studio自带的Java JDK发现出错。
The operation couldn’t be completed. Unable to locate a Java Runtime.

原因是因为Android Studio的JDK没有javap，所以只能用brew安装了。

brew install Java
